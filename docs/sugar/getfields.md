# Examples
```matlab
getfields(struct('a',1,'b',2,'c',3), ["a","b"])
getfields(struct('a',1,'b',2,'c',3), ["a";"b"])
getfields(struct('a',1,'b',2,'c',3), {"a";"b"})
getfields(struct('a',1,'b',2,'c',3), {'a','b'})
```
