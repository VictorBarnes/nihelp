# COUNTNATURALS Count occurences of natural numbers (positive integers)
 
# Examples
```matlab
N = countNaturals(1:5)
N = countNaturals(-1:5)
N = countNaturals(magic(2))
N = countNaturals(magic(2).^2)
```
