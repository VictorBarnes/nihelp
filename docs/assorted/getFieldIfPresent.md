# GETFIELDIFPRESENT checks if a struct contains a field; if so, returns it
 
# Examples
```matlab
getFieldIfPresent(struct('a', 1, 'b', 2), 'a')
getFieldIfPresent(struct('a', 1, 'b', 2), 'c')
```
