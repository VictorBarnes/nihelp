# CORRFC Applies r-to-z transform then calculates Pearson correlations between FC matrices
 
# Examples
```matlab
corrFC(randFC(5,[],[],1), randFC(5,[],[],2))
corrFC(randFC(5,[],[],1), randFC(5,[],[],2), randFC(5,[],[],3))
```
