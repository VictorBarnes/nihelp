# RANDFC Generates random FC matrix with specified size, mean, and stnadard deviation
 
# Examples
```matlab
randFC(5)
randFC(6,0.5)
randFC(7,0.5,0.01)
randFC(4,[],[],1)
randFC(4,[],[],1)
randFC(4,[],[],2)
```
