# VOLUMETRICNEIGHBORS Finds the data in a volume V adjacent to a mask
 
# Examples
```matlab
m=reshape(idx2mask(14,27),[3 3 3]); V=reshape(1:27,[3 3 3]); volumetricNeighbors(m,V)
```
