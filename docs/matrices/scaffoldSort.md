# SCAFFOLDSORT Reorder data according to a provided scaffold
 
# Examples
```matlab
scaffoldSort([10 50 30 20], [6 8 7 9])
scaffoldSort([10 50 30 20], [4 3 2 1])
scaffoldSort([4 3 2 1], [10 50 30 20])
```
