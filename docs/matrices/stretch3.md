# STRETCH3 Converts 4-d volumetric time series data to 2-d vectorised time series
 
# Examples
```matlab
V = magic(3)+permute([0 0 0],[1 3 2])+permute(1:10,[1 4 3 2]); out = stretch3(V);
```
