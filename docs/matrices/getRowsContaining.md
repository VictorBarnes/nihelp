# GETROWSCONTAINING Gets rows in from matrix containing specified values
 
# Examples
```matlab
X = (0:6)' + (1:3); getRowsContaining(X,5)
X = (0:6)' + (1:3); getRowsContaining(X,[5,6,7])
```
