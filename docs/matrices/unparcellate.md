# UNPARCELLATE Converts data from parcel/ROI level to vertex level
 
# Examples
```matlab
  unparcellate([0 1 1 0 2 2 0 3 3 0]', [1 4 5]')
  unparcellate([0 1 1 0 2 2 0 4 4 0]', [1 4 3 5]')
  unparcellate([0 1 1 0 2 2 0 4 4 0]', [1 4 3 5]', 0)
```
